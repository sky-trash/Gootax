# Блог на Yii2 - Тестовое задание

Полнофункциональный блог на Yii2 Framework с системой постов, комментариев и админ-панелью.

## Функционал

### Публичная часть
- Главная страница с последними записями
- Страница просмотра поста с комментариями
- Система комментариев
- Адаптивный дизайн

### Админ-панель
- Управление постами (CRUD)
- Создание, редактирование, удаление записей
- Управление статусами (черновик/опубликовано)

### Технические особенности
- SQLite база данных
- Миграции
- Pretty URL
- Аутентификация пользователей
- PSR стандарты кодирования

## Технологии

- Yii2 PHP Framework
- SQLite
- Bootstrap
- PHP 8.0+
